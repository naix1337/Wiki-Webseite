# CLI Reference

| Befehl | Beschreibung              | Beispiel              |
| ------ | ------------------------- | --------------------- |
| `init` | Initialisiert Projekt     | `tool init my-app`    |
| `dev`  | Startet lokalen Server    | `tool dev --open`     |
| `build`| Baut Produktionsartefakte | `tool build --clean`  |
